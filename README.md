# FervoGear Black Friday Landing Page

High-converting Black Friday landing page for FervoGear's Double Layer SFI-5 Nomex Race Suit.

## Features

- **Conversion-Optimized Design**: Direct response copywriting with click funnel psychology
- **Countdown Timer**: Creates urgency with live countdown to deal expiration
- **Value Stack**: Clearly shows $451 in total savings
- **Social Proof**: Real driver testimonials and gallery
- **Scarcity Messaging**: Limited to 47 suits with visual progress bar
- **Multiple CTAs**: Strategic placement throughout the funnel
- **5-Step Process**: Clear visualization of the custom suit journey
- **Mobile-First**: Fully responsive design
- **Fast Performance**: Optimized Next.js build with image optimization

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Server Components

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
